    - Windows: https://repo.elemento.cloud/app/Elemento_Cloud_App.exe
    - Mac ARM: https://repo.elemento.cloud/app/Elemento_Cloud_App.dmg
    - Mac Intel:https://repo.elemento.cloud/app/Elemento_Cloud_App_intel.dmg
